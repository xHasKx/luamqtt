next_packet_id ([curr])
Returns the next Packet Identifier value relative to given current value. If current is nil - returns 1 as the first possible Packet ID. For MQTT v3.1.1: 2.3.1 Packet Identifier, For MQTT v5.0: 2.2.1 Packet Identifier.

Parameters:

  • curr number - current Packet ID value (optional)

Returns:

    number next Packet ID value
